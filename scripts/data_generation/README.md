# Dossier `scripts/data_generation`

Scripts pour générer les jeux de données synthétiques (users, consommation, recharges, zones).

Utilisation:
- Lancer `python scripts/generate_all_data.py` pour recréer tous les CSV dans `data/raw/`.

Vérifier les paramètres dans les scripts individuels si vous avez besoin d'autres distributions.
