# Dossier `scripts`

Contient les scripts de génération de données, ETL et utilitaires.

Sous-dossiers importants:
- `scripts/data_generation/` : génération des CSV
- `scripts/etl/` : scripts d'ingestion et loaders
- `scripts/simulation/` : simulateur de recharges et CLI
- `scripts/utils/` : utilitaires (performance, helpers)

Exécution locale : activer le venv puis lancer les scripts depuis la racine du repo.
